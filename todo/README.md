To set-up todo:

1. Copy todo.sh to /usr/local/bin/ using sudo and make executable (sudo chmod +
   x todo.sh)
2. Set-up ~/.todo/ folder and add the following shortcuts
- config -> ~/wsl-config/todo/config
- todo.actions.d -> /mnt/c/Users/[username]/OneDrive/todo.txt/todo.actions.d
- todo.txt -> /mnt/c/Users/[username]/OneDrive/todo.txt/
